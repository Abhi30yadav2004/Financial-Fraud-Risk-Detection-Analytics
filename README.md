# 🚀 Financial Fraud Detection System

## 📌 Overview

Developed a machine learning-based system to detect fraudulent transactions and created an interactive Power BI dashboard to analyze fraud patterns and high-risk behaviors.

---

## 🛠️ Tools Used

* Python (Pandas, Scikit-learn)
* Power BI
* Jupyter Notebook

---

## 🤖 Model

* Algorithm: Logistic Regression
* Accuracy: ~99.6%
* Evaluated using Confusion Matrix

---

## 📊 Key Insights

* Fraud rate ~8% across transactions
* High-value transactions (>₹50,000) show higher fraud risk
* Foreign and night transactions are more prone to fraud

---

## ⚠️ Risk Analysis

* Created **High Risk Rate (35–45%)** using behavioral conditions
* Compared High Risk vs Actual Fraud to identify early warning signals

---

## 📈 Dashboard Highlights

* Fraud Rate & High Risk Rate by location
* Fraud vs Risk comparison
* Transaction amount-based fraud analysis

---

## 💡 Business Impact

* Helps identify high-risk transactions early
* Supports fraud prevention strategies (OTP, alerts, monitoring)
